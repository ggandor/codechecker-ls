# codechecker-ls

Work in progress diagnostic language server for
[CodeChecker](https://github.com/Ericsson/codechecker). Early testing is
appreciated, but expect breaking changes without notice.

## Getting started

### Requirements

* [build](https://build.pypa.io/en/latest/installation.html)

### Installation

After cloning the repository, run:

```
python -m build
pip install --user dist/codechekcer-ls-<version>.tar.gz
```

The server executable will be available as `codechecker-ls` on the path.
